# Конфігурації планет. Рівняння синодичного руху

**Конфігурації планет** — це характерні взаємні розміщення планет, Землі та Сонця у просторі. Вони визначають умови видимості небесних тіл із Землі, зокрема те, в який час доби об'єкт з'являється над горизонтом і коли він найближче до нас (найяскравіший).

## Розподіл планет за орбітами

Для земного спостерігача планети Сонячної системи поділяються на дві групи залежно від того, де лежить їхня орбіта:

- **Нижні (внутрішні) планети:** Меркурій та Венера. Їхні орбіти розташовані ближче до Сонця, ніж земна, тому на небі вони ніколи не віддаляються далеко від сонячного диска.
- **Верхні (зовнішні) планети:** Марс, Юпітер, Сатурн, Уран, Нептун. Їхні орбіти лежать за межами орбіти Землі.

## Основні конфігурації

| Група планет | Назва конфігурації            | Опис розташування та умови видимості                                                                                                                                     |
| ------------ | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Нижні**    | Сполучення (нижнє / верхнє)   | Планета на одній прямій із Сонцем (між Землею і Сонцем, або за Сонцем). **Невидима**, бо губиться в сонячних променях.                                                   |
|              | Елонгація (східна / західна)  | Планета відхиляється від Сонця на максимальний кут (для Венери $\approx47^\circ$, Меркурія $\approx28^\circ$). **Найкращий час** для ранкових або вечірніх спостережень. |
| **Верхні**   | Сполучення                    | Планета перебуває за Сонцем на одній лінії із Землею. **Невидима**.                                                                                                      |
|              | Протистояння                  | Земля перебуває рівно між Сонцем і планетою. **Найкращий час спостережень**: планету видно всю ніч, і вона найближче до Землі.                                           |
|              | Квадратура (східна / західна) | Кут між напрямками на Сонце і планету із Землі становить рівно $90^\circ$.                                                                                               |

## Сидеричний та синодичний періоди

Оскільки Земля не стоїть на місці, ми спостерігаємо рух інших планет із рухомої платформи. Тому астрономи розрізняють два періоди:

- **Сидеричний період ($T$):** Істинний час одного повного оберту планети навколо Сонця відносно нерухомих зір (зоряний рік планети).
- **Синодичний період ($S$):** Проміжок часу між двома послідовними однаковими конфігураціями (наприклад, час від одного протистояння Марса до наступного).

## Рівняння синодичного руху

Ці рівняння математично пов'язують синодичний період ($S$), сидеричний період планети ($T$) та сидеричний період Землі ($T_\oplus$, що дорівнює 1 року). Вони виводяться з різниці кутових швидкостей тіл.

Для **нижніх планет** (рухаються по орбіті швидше за Землю, $T<T_\oplus$):

$$\frac{1}{S} = \frac{1}{T} - \frac{1}{T_\oplus}$$

Для **верхніх планет** (рухаються повільніше за Землю, $T>T_\oplus$):

$$\frac{1}{S} = \frac{1}{T_\oplus} - \frac{1}{T}$$

_Де всі періоди обов'язково виражаються в однакових одиницях (наприклад, у земних роках або добах)._

## Підсумок

Геометрія орбіт зумовлює те, що нижні планети ми можемо бачити лише вранці або ввечері поблизу Сонця (під час елонгацій), тоді як верхні планети можуть спостерігатися на протилежному боці неба вночі (під час протистоянь). Рівняння синодичного руху є базовим інструментом для розрахунку того, коли саме настане наступний сприятливий період для спостереження за будь-якою планетою.
